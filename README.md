# Manuskrip Perubatan Melayu — Template GitHub Pages (Bahasa Melayu)

Ini ialah template GitHub Pages (Jekyll) untuk menyimpan 10 manuskrip perubatan tradisional Melayu dalam format Markdown.

Cara guna ringkas:
1. Salin semua fail dan struktur ke repositori baru (contoh nama: manuskrip-perubatan-melayu).
2. git add .; git commit -m "Initial: Jekyll pages & manuskrip"
3. git push origin main
4. GitHub → Settings → Pages → Source: main branch / (root) → Save.
5. Tunggu beberapa minit. Laman akan muncul di: https://<USERNAME>.github.io/<REPO>/

Amaran:
- Kandungan bertujuan dokumentasi budaya sahaja dan bukan nasihat perubatan profesional. Jika gejala serius, dapatkan bantuan kesihatan.

Jika mahu saya bantu:
- Sediakan fail HTML untuk setiap manuskrip (bukan menggunakan Jekyll).
- Tambah fungsi carian, kategori, atau terjemah ke Bahasa Inggeris.
- Tambah LICENSE (MIT/CC-BY).
